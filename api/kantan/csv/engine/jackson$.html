kantan.csv.engine

jackson 

object jackson

Provides CSV reader and writer engines using jackson.csv.

Importing kantan.csv.engine.jackson._ will replace default engines by the jackson-backed ones. If you need to tweak how jackson.csv behaves, however, you can handcraft engines though readerEngineFrom and writerEngineFrom - all you need is a function that knows how to turn a column separator character in an instance of CsvSchema.
